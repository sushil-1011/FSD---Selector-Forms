# CSS Selectors – FSD Practice

## Project Overview

This repository contains simple HTML files created to understand different types of CSS selectors in Full Stack Development (FSD).

The purpose of this project is to practice and clearly understand how CSS selectors work and how they are used to style HTML elements.

Each file demonstrates a specific type of CSS selector.


## Objectives

- Learn different types of CSS selectors
- Understand how selectors target HTML elements
- Practice styling using various selector techniques
- Improve frontend development basics


## Technologies Used

- HTML5
- CSS3

## File Description

### 1. simple_selector.html

Demonstrates basic selectors such as:
- Element selector
- Class selector
- ID selector
- Universal selector
- Group selector

These are the most basic and commonly used CSS selectors.


### 2. attribute_selector.html

Demonstrates how to style elements based on their attributes.

Examples include:
- Selecting elements with specific attributes
- Selecting elements with specific attribute values


### 3. combinatory_selector.html

Demonstrates combinator selectors that define relationships between elements.

Includes:
- Descendant selector
- Child selector
- Adjacent sibling selector
- General sibling selector


### 4. pseudo_class.html

Demonstrates pseudo-classes that style elements based on their state.

Examples:
- hover
- active
- focus
- first-child
- nth-child


### 5. pseudo_element_selector.html

Demonstrates pseudo-elements used to style specific parts of an element.

Examples:
- first-line
- first-letter
- before
- after
